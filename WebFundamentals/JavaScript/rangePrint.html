Range Print

Create a function that can take a start point, end point, and skip amount, to print all numbers in a range (exclusive of final end point number).

Bonus:

  • Make sure it works for negative numbers
  • If the user doesn't pass a skipAmount, make it default to 1
  • If the user doesn't pass an endPoint, make it start at 0 and end at the first

Open the console in your browser/ interpreter to view the answer.
