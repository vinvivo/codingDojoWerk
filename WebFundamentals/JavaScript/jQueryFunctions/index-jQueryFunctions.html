Skim through the documentation quickly and casually play around to get familiar with only the following jQuery methods. Once you have spent 4-5 hours on this assignment, move on as you'll get to learn more about how you can use these jQuery methods to build some amazing things later on. The wireframe on the right shows only 3 functions being tested. In this assignment, we want you to add more and test all 18 jQuery functions shown below.

  • .addClass

    SYNTAX: $("target").addClass("className[ className className]")

    This method adds the specified class(es) to each element in the set of matched target elements.

    Click the button to add the class 'highlight' to this line.

  • .after

    SYNTAX: $("target").after(content[, content])

    This method inserts content, specified by the above parameter content, after the target element tags (i.e., after the target element's closing tags). (Different from .append below.)

    Click the button to insert text after this section's h2 header to complete the title.

  • .append

    SYNTAX: $("target").append(content[, content])

    This method inserts content, specified by the above parameter content, as the last child of the target element (i.e., within the target element tags right before the closing tags). (Different from .after above.)

    Click the button to append text at the end (but still within) this section's h2 header to create a medical emergency.

  • .attr

    SYNTAX: $("target").attr("attribute")

    This method gets the value of an attribute for the first element in a set of matched target elements, or sets one or more attributes for every matched elements.

    Click the button to display the src attribute of the jQuery logo.
