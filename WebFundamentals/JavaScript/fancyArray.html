Fancy Array

Normally, if you print an array such as ["James", "Jill", "Jane", "Jack"], you will see the following:

[ "Christian", "Ciara", "Damian", "Marian" ]

Let's make it look fancy! Write a function that will make it print like:

0 -> Christian

1 -> Ciara

2 -> Damian

3 -> Marian
