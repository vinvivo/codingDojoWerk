Assignment: Pokemon

Create a website that has all the pictures of the original Pokemon (Pokemon #1 ~ Pokemon #151). Use the PokeAPI to get the images. You can find the image of the first Pokemon on http://pokeapi.co/media/img/1.png, second Pokemon on http://pokeapi.co/media/img/2.png, third Pokemon on http://pokeapi.co/media/img/3.png, and so on. (Copy and paste those URLs. Hotlinking has been prohibited by pokeapi.)

You can list out 151 images or you can use a for loop (use a for loop).
